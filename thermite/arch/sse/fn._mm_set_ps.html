pub unsafe fn _mm_set_ps(a: f32, b: f32, c: f32, d: f32) -> __m128
This is supported with target feature sse only.

Construct a __m128 from four floating point values highest to lowest.

Note that a will be the highest 32 bits of the result, and d the lowest. This matches the standard way of writing bit patterns on x86:

 bit    127 .. 96  95 .. 64  63 .. 32  31 .. 0
       +---------+---------+---------+---------+
       |    a    |    b    |    c    |    d    |   result
       +---------+---------+---------+---------+

Alternatively:

let v = _mm_set_ps(d, c, b, a);
